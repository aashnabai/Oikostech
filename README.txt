Deployment (GitHub Pages):
1. Create a repository on GitHub (if not already).
2. Upload the files at repository root (do not place inside a subfolder).
3. In Settings > Pages set Branch: main and Folder: / (root).
4. Wait ~1 minute and open https://YOUR_USERNAME.github.io/REPO_NAME/

Notes:
- Booking and Feedback use Google Forms. They open in a new tab (Google may require login for file uploads).
- To change form URLs edit index.html/book.html/feedback.html and replace the two form links.
- Colors: primary green #0C5037, gold accent #D9A441, light cream backgrounds.
